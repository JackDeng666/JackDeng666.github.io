1. 项目初始化​

首先，要创建项目，封装 vue 的插件，初始化 vue 工程一般用： vue create [项目名]，但是我们要写的是一个简单的 vue 组件，不需要依赖那么多而庞大的配置，所以，这里我们用简版的 webapck 配置模板：

vue init webpack-simple [项目名]
